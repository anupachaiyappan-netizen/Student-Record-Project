# Student Record Management System

## Project Description
This project allows the user to add student details, view all records, search for a student and exit.
The project demonstrates the use of basic Python programming concepts.

---

## Features
- Add student details
- View all student records
- Search student by ID
- Store data using file handling


---

## Technologies Used
- Python
- Jupyter Notebook

---

## Concepts Used and Mapping

- Lists  
  Used to store multiple student records in the program.

- Dictionaries  
  Each student record is stored using a dictionary with ID, name and course.

- Functions  
  Separate functions are used for adding, viewing and searching students.

- Conditional Statements  
  if-elif-else statements are used to handle menu choices.

- Loops  
  A while loop is used to display the menu repeatedly until exit.

- File Handling  
  Student details are stored permanently in a text file using open().

- Exception Handling  
  Used to handle invalid inputs entered by the user.

---
